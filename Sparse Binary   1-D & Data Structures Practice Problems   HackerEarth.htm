Sparse Binary

A sparse binary integer is an integer whose binary representation does not contain any consecutive 1s.
Given an integer N in the input, find its nearest sparse binary number greater than or equal to N. 


Input:

The first line of input contains an integer T denoting the number of test cases. The first line of each test case is number 'N'

Output:

Print the next Sparse binary number in its decimal form if the input number is not sparse binary else print the number itself.

Constraints:
1 T 100
1 n 10^12

SAMPLE INPUT
2
3
5
SAMPLE OUTPUT
4
5
Explanation

Binary representation of 3 is  '11' next sparse binary number is '100'. so output will be 4.

Binary representation of 5 is '101' its a sparse binary number so output is 5 

Time Limit: 1.0 sec(s) for each input file.
Memory Limit: 256 MB
Source Limit: 1024 KB
Marking Scheme: Marks are awarded when all the testcases pass.
